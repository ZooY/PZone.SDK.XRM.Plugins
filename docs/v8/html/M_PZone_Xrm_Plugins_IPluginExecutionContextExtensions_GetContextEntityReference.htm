IPluginExecutionContextExtensionsGetContextEntityReference - метод

Получение ссылки на сущность из контекста плагина.

Пространство имён:  PZone.Xrm.Plugins
Сборка:  PZone.SDK.Xrm.Plugins (в PZone.SDK.Xrm.Plugins.dll) Версия: 8.2.1.0 (8.2.1.0)
Синтаксис
C#
Копировать
public static EntityReference GetContextEntityReference(
	this IPluginExecutionContext context
)

Параметры

context
Тип: IPluginExecutionContext
Контекст плагина.

Возвращаемое значение

Тип: EntityReference
Метод возвращает ссылку на сущность из контекста плагина, если событие предусматривает передачу ссылки.

Примечание об использовании

В Visual Basic и C# этот метод можно вызывать как метод экземпляра для любого объекта типа IPluginExecutionContext. При вызове метода для экземпляра следует опускать первый параметр. Дополнительные сведения см. в разделе Методы расширения (Visual Basic) или Методы расширения (Руководство по программированию в C#).
Исключения
ИсключениеУсловие
Exception Ссылка на сущность не доступна в контексте данного события.
Заметки
Ссылки передаются, например, при событиях DeleteRequest, AssociateRequest, DisassociateRequest и др.
Примеры
Копировать
public class MyPlugin : IPlugin
{
    public void Execute(IServiceProvider serviceProvider)
    {
        ...
        var context = serviceProvider.GetContext();
        var entityRef = context.GetContextEntityReference();
        ...
    }
}
